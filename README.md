# Housing Price Prediction

This project predicts California housing prices using Machine Learning.

## Features
- Data Preprocessing
- Missing Value Handling
- One Hot Encoding
- Standardization
- Random Forest Regressor
- Model Saving with Joblib

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
