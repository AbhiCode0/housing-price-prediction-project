# 🏠 Housing Price Prediction

A Machine Learning project that predicts California housing prices using Random Forest Regression and Scikit-learn.

## 📌 Features

- Data Preprocessing
- Handling Missing Values
- Feature Scaling
- One-Hot Encoding
- Random Forest Regressor
- Model Saving using Joblib
- Prediction on New Data

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Joblib

## 📂 Project Structure

```
housing-price-prediction/
│── main.py
│── housing.csv
│── requirements.txt
│── README.md
│── LICENSE
│── .gitignore
```

## 🚀 Installation

```bash
git clone https://github.com/AbhiCode0/housing-price-prediction-project.git
cd housing-price-prediction-project
pip install -r requirements.txt
```

## ▶️ Run

```bash
python main.py
```

## 📈 Machine Learning Workflow

- Load Dataset
- Data Preprocessing
- Train/Test Split
- Feature Engineering
- Train Random Forest Model
- Save Model
- Predict Housing Prices
